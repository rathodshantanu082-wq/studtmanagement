# Simple Student Management System

A basic Python console application to manage student records including adding, viewing, searching, updating, and deleting information. Data is saved in a text file for persistence.

---

## Features

- Add new student details (name, roll number, branch, marks)  
- View all students in a tabular format  
- Search students by name or roll number  
- Update student marks  
- Delete student records  
- Show class statistics (total students, average marks, top marks)  
- Data is automatically saved in `students.txt`

---

## Requirements

- Python 3.x installed on your system

## Sample Usage

- Adding a new student: Input student name, roll no, branch, and marks  
- View all students: Displays a list with ID, name, roll no, branch, and marks  
- Search student: Enter part of the name or roll no to find student(s)  
- Update marks: Provide roll no and new marks to update  
- Delete student: Provide roll no to remove student record  
- Show statistics: View total students, average marks, and highest marks  

---

## Project Structure

- `simple_student.py` — Main Python program file  
- `students.txt` — Data file storing student records (created automatically)
